1. Snapshot – Simplified vs Traditional

Aspect Simplified (Focus) Traditional (Reference)

Character count ~8,000 total; ~3,500 for 99% daily use ~13,000 total; ~4,800 for 99% daily use
Typical font Noto Sans SC, Source Han Sans Noto Sans TC, JhengHei
Typing Pinyin → pick character Zhuyin or Pinyin → pick character

3. Universal Stroke-Order Rules

  1. Top → bottom
  2. Left → right
  3. Outside → inside

4. Daily Study Recipe

  • Learn: 10–20 new Hanzi per day via spaced-repetition (Anki / Skritter / Pleco).
  • Read aloud: every card to nail tones.
  • Type daily: switch phone keyboard to Pinyin (Simplified) and chat.
